# House Rent Prediction using Linear Regression

## Project Overview

This project predicts house rental prices using Machine Learning and Linear Regression based on features like:

- Area (sqft)
- Bedrooms
- Locality
- Property Type
- Amenities
- Age of Property

## Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Outlier Handling
4. Encoding Categorical Variables
5. Feature Scaling
6. Linear Regression Model Training
7. Model Evaluation

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Model Performance

- R² Score: 0.69
- MAE: 6967
- RMSE: 25572

## Key Insights

- Area and locality strongly influence rental prices.
- Log transformation improved target distribution.
- Linear Regression captured major pricing trends successfully.
